## ðŸ“ž Usage Support

### Common Questions

**Q: What's the minimum age for educational content?**
A: Educational elements can start at birth (lullabies teach language rhythm), but formal academic content (ABCs, numbers) most appropriate ages 2+.

**Q: Can one song work for multiple ages?**
A: Songs for broader age ranges (e.g., 2-4 years) possible by:
- Keeping vocabulary simple
- Using moderate tempo
- High repetition
- Clear, simple concepts
But targeted age ranges more effective.

**Q: How much repetition is too much?**
A: See repetition ratios in AGE_SPECIFIC_GUIDELINES.json. Never "too much" for young children - they crave repetition.

**Q: Is it okay to use minor keys for children?**
A: Yes, appropriate for:
- Some lullabies (calming, mysterious)
- Contemplative moods
- Emotional depth songs (ages 4+)
- Cultural music that traditionally uses minor
But major keys generally preferred for young children.

**Q: How do I balance education with fun?**
A: Best educational songs don't feel like teaching:
- Catchy melodies
- Movement integration
- Playful delivery
- Story or game elements
- Musical "fun" comes first, learning is embedded
